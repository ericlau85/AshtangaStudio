体式

समस्थितिः samasthitiḥ

sama(相等的) + sthitiḥ(稳定)

सूर्य नमस्कार sūrya namaskāra

sūrya(太阳) + namaskāra(礼拜)

पादाङ्गुष्ठासन pādāṅguṣṭhāsana

pāda(脚) + aṅguṣṭha(脚趾) + āsana

पादहस्तासन pāda hastāsana

pāda(脚) + hasta(手) + āsana

उत्थित त्रिकोणासन utthita trikoṇāsana

utthita(伸展的) + tri(三) + koṇa(角) + āsana

उत्थित पार्श्वकोणासन utthita pārśvakoṇāsana

utthita(伸展的) + pārśva(侧面) + koṇa(角) + āsana

प्रसारित पादोत्तानासन prasārita pādottānāsana

prasārita(展开的) + pāda(脚) + uttāna(强烈伸展) + āsana

पार्श्वोत्तानासन pārśvottānāsana

pārśva(侧面) + uttāna(强烈伸展) + āsana

उत्थितहस्त पादाङ्गुष्ठासन utthita hasta pādāṅguṣṭhāsana

utthita(伸展的) + hasta(手) + pāda(脚) + aṅguṣṭha(脚趾) + āsana

अर्धबद्ध पद्मोत्तानासन ardha baddha padmottānāsana

ardha(一半) + baddha(束缚的) + padma(莲花) + uttāna(强烈伸展) + āsana

उत्कटासन utkaṭāsana

utkaṭa(强大的) + āsana

वीरभद्रासन vīrabhadrāsana

vīrabhadra(维拉巴德拉「战士」) + āsana

दण्डासन daṇḍāsana

daṇḍa(杖) + āsana

पश्चिमोत्तानासन paścimottānāsana

paścima(西方) + uttāna(强烈伸展) + āsana

पूर्वोत्तानासन pūrvottānāsana

pūrva(东方) + uttāna(强烈伸展) + āsana

अर्धबद्धपद्म पश्चिमोत्तानासन ardha baddha padma paścimottānāsana

ardha(一半) + baddha(束缚的) + padma(莲花) + paścima(西方) + uttāna(强烈伸展) + āsana

त्रियङ्गमुखैकपाद पश्चिमोत्तानासन triyaṅga mukhaikapāda paścimottānāsana

tri(三) + aṅga(肢体) + mukha(脸) + eka(一) + pāda(脚) + paścima(西方) + uttāna(强烈伸展) + āsana

जानुशीर्षासन jānu śīrṣāsana

jānu(膝) + śīrṣa(头) + āsana

मरीच्यासन marīcyāsana

marīci(马里奇「摩利支」) + āsana

नावासन nāvāsana

nāva(船) + āsana

कुक्कुटासन kukkuṭāsana

बद्ध कोणासन baddha koṇāsana

supta(卧) + koṇa(角) + āsana

सुप्त पादाङ्गुष्ठासन supta pādāṅguṣṭhāsana

supta(卧) + pāda(脚) + aṅguṣṭha(脚趾) + āsana

उभय पादाङ्गुष्ठासन ubhaya pādāṅguṣṭhāsana
